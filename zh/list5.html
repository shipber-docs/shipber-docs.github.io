# 为什么返回报错Ship Address not found？

常见错误返回报文示例：

{ "status_code": 400,"message": "Ship Address not found ."}

# 解决方法：

1、uuid填写错误，调用/v1/open-api/ship-from接口进行获取正确的uuid。具体可以跳转至：发件地址open in new window查看。
